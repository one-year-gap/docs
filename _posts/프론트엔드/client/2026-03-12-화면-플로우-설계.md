    **3) 고객 페이지 UI**
  **0. 스플래시 페이지**

- 화면 UI

![](/docs/assets/images/uploads/%EC%8A%A4%ED%94%8C%EB%9E%98%EC%8B%9C2.png)

- 페이지 소개
        - Holliverse 서비스에 첫 접속 시 보이는 화면입니다.
        - 서비스를 시작하기 전, Holliverse의 이미지와 분위기를 미리 보여줍니다.
  **1. 회원가입/로그인 페이지**
- 화면 UI

![](/docs/assets/images/uploads/%EB%A1%9C%EA%B7%B8%EC%9D%B8-%ED%9A%8C%EC%9B%90%EA%B0%80%EC%9E%85.png)

- 페이지 소개
        - 스플래시 페이지에서 연결되는 화면 로그인 화면입니다.
        - 저희 Holliverse는 폼 로그인, 구글 소셜 로그인 방법을 제공합니다.
        - 회원가입 후 약관동의 페이지로 이동하며, 동의할 경우 온보딩 페이지로 이동합니다. [온보딩 페이지 추가 예정]
        - 회원가입을 마친 후에는 로그인 페이지로 돌아와, 사용자의 정보를 입력하여 메인 페이지로 진입할 수 있습니다.
  **2. 메인 페이지**
- 화면 UI

![](/docs/assets/images/uploads/%EB%A9%94%EC%9D%B8.png)

- 페이지 소개
        - 로그인 후 처음 진입하게되는 대시보드 페이지입니다.
        - 사용자의 이름, 캐릭터 유형과 같은 기본적인 정보를 상단에서 확인할 수 있습니다.
        - 사용자의 데이터/통화/문자 사용량을 확인할 수 있습니다.
        - 카테고리 칩, 배너, 바텀 네비게이션 바를 통해 다양한 페이지로 이동할 수 있습니다.
  **3. 상품 페이지**
- 화면 UI

![](/docs/assets/images/uploads/%EC%83%81%ED%92%88.png)

- 페이지 소개
        - 상품 페이지는 가장 좌측에 있는 추천 칩 활성/비활성화를 통해 조회되는 상품의 방식을 변경할 수 있습니다.
        - 추천 칩 비활성화 상태 - 전체 상품 조회
                - 실제 LG U+에서 제공하는 요금제 상품을 카테고리별로 조회할 수 있습니다.
                - 카테고리 종류는 5G 요금제, 스마트워치/태블릿, 인터넷, IPTV, 부가서비스로 총 5가지 입니다.
                - 요금제 리스트의 항목을 선택하여 상세 정보를 확인할 수 있으며, 현재 가입 중인 사용자의 요금제와 비교해보고 변경할 수 있습니다.
        - 추천 칩 활성화 상태 - 추천 상품 조회
                - Holliverse는 사용자의 서비스 이용 정보, 기본 정보를 취합하여 해당 사용자에게 알맞는 요금제를 추천해줍니다.
                - 추천 상품 조회 화면에서는 각 상품이 추천되는 이유에 대한 설명을 함께 확인할 수 있습니다.
  **4. 유형 페이지**
- 화면 UI
    - 페이지 소개
        - Holliverse에서는 사용자의 성향을 분석하여 캐릭터를 배정해줍니다.
        - 유형 페이지에서는 사용자의 캐릭터 유형, 서비스 이용 패턴, 분석 지표를 조회할 수 있습니다.
  **5. 마이 페이지**
- 화면 UI
- 페이지 소개
        - 사용자의 가입 정보를 조회 및 수정할 수 있습니다.
        - 자주 묻는 질문, 약관 및 정책을 조회할 수 있습니다.
        - 로그아웃을 할 수 있습니다.
